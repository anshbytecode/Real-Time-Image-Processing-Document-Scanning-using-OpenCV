📄 Real-Time Image Processing and Document Scanning using OpenCV
A Computer Vision project developed using Python and OpenCV that automatically detects and scans documents in real time. The application captures a document from a webcam feed (or an image file), identifies its boundaries, performs perspective transformation, and generates a clean, high-quality scanned output similar to mobile document scanning applications.

🚀 Features
Real-time document detection using a webcam
Automatic contour detection and extraction
Perspective correction (bird's-eye view transformation)
Adaptive thresholding for a scanned-document effect
Support for both webcam and image file inputs
Interactive threshold adjustment using trackbars
Image preprocessing using grayscale conversion and Gaussian blur
Edge detection using the Canny algorithm
🛠️ Technologies Used
Python
OpenCV
NumPy
📸 Output Stages
The application displays the complete document scanning pipeline:

Original Image
Grayscale Image
Edge Detection
Contour Detection
Largest Contour Detection
Perspective Warping
Warped Grayscale Image
Adaptive Thresholded Scan
